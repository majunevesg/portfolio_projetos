# Portfólio de Projetos – Maria Julia

Este repositório contém o código do meu portfólio de projetos em análise de dados e estatística.

O objetivo do portfólio é apresentar alguns projetos desenvolvidos com foco em:

- análise exploratória de dados
- visualização de dados
- indicadores e estatística aplicada
- uso de bases públicas ou dados simulados

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- GitHub Pages

## Projetos apresentados

Alguns projetos presentes no portfólio:

- Análise de dados do Disque 100
- Exploração de bases públicas
- Visualização de dados e indicadores
