Projetos

Esta seção reúne alguns projetos desenvolvidos por mim. A ideia é apresentar, de forma resumida, o problema analisado, as ferramentas utilizadas e os principais resultados obtidos. Os projetos foram desenvolvidos utilizando dados disponibilizados publicamente ou dados simulados, com o objetivo de reproduzir situações próximas a problemas reais de análise de dados. Em cada projeto busco estruturar o processo de análise de forma completa, incluindo etapas como compreensão do problema, tratamento e organização dos dados, análise exploratória, geração de visualizações e interpretação dos resultados. O objetivo é demonstrar a aplicação prática de métodos estatísticos e técnicas de análise de dados para gerar insights relevantes e apoiar a tomada de decisão.
